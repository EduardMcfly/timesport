## Installation

1. Clone this repository: `git clone git@EduardMcfly/timesport.git`.
2. `cd` into `timesport`: `cd timesport`
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running

```
python manage.py runserver
```
